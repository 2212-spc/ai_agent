# Vue版本快速开始 🚀

## ⚡ 一键启动（推荐）

**双击运行**：`启动Vue开发.bat`

或者使用命令行：

```bash
cd "d:\PycCharm Code\AI-Agent\ai_agent\frontend-vue"
npm run dev
```

浏览器访问：http://localhost:5173/

---

## 📋 前置要求

- ✅ Node.js 18+ （下载：https://nodejs.org/）
- ✅ 后端服务已启动（`http://127.0.0.1:8000`）

## 🎯 核心功能测试

### 1. 账号记忆隔离 🔒

- 开启**全局记忆**开关
- 账号A说："我叫张三"
- 切换账号B，问："我叫什么？"
- 结果：账号B看不到账号A的记忆 ✅

### 2. 深度思考模式 💭

- 开启**深度思考**开关
- 问："解释量子纠缠"
- 结果：显示多个详细思考步骤 ✅

### 3. 新建对话 🆕

- 点击"新建对话"
- 发送消息
- 结果：正常显示，不会空白 ✅

### 4. 上网搜索工具 🔧

- 单智能体模式下
- 问："今天天气怎么样？"
- 结果：能调用搜索工具，返回实时信息 ✅

### 5. 重新生成答案 🔄

- 发送问题，等待回答
- 点击答案下方的"重新生成"按钮
- 结果：重新生成新的答案 ✅

## 🐛 遇到问题？

1. **控制台报错**：F12 → Console 查看
2. **后端未启动**：确保 http://127.0.0.1:8000 可访问
3. **依赖问题**：删除 `node_modules`，重新 `npm install`
4. **端口被占用**：修改 `vite.config.js` 改端口
